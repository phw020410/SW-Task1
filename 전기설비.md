전기 설비에 필요한 설계 작성

- 필요한 콘센트 개수 파악
    - 3층에 각 층마다 2개의 동이 있음
    - 방 개수 거실 + 주방 1개, 방 1개, 화장실 1개
    - 콘센트 개수 : 거실 1개 + 주방 1개 + 방 1개 + 화장실 1개 = 4개
    - 각 집마다 4개 필요. 즉, 3층 * 2집 * 4 = 24개

- 주택의 전기가 필요한 편의시설(복도 전등, 엘리베이터) 파악
    - 층마다 중앙 전등
    - 1층 공동 현관문 보안 설치
    - 1층 공동 현관 앞 전등등
    - 수도
    3층이기에 엘리베이터 X

- 주택 전체 관리하는 전기 시설 위치
    - 주택 지하 1층
